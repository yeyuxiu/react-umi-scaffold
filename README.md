# react-umi-scaffold

react-umi 搭建的脚手架

## 版本

> @types/react: ^18.0.33 @umijs/max: ^4.4.7 antd: ^5.4.0 node: 20.18.1

## 启动命令

```bash
pnpm install

pnpm start
```

### 初始化

1. `config/config.ts` `src/app.ts`的基本配置
2. layouts 基本布局
3. 丰富`utils`工具函数
4. 补充`compents`组件库
5. 完善登录、普通请求
6. `@umijs/max` 配置使用

[ahooks](https://ahooks-v2.js.org/zh-CN/hooks/async) [umi4](https://umijs.org/docs/max/request)
